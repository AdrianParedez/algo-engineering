# Code of Conduct

## Purpose

This repository is for rigorous, implementation-oriented work on algorithm engineering. Technical disagreement is expected. Personal hostility is not.

## Expected Behavior

- Argue from evidence, measurements, source material, and reproducible examples.
- Keep review feedback specific, technical, and actionable.
- State uncertainty explicitly when a claim is incomplete or provisional.
- Respect contributors across experience levels, backgrounds, and domains.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation.
- Personal attacks, insults, or bad-faith pile-ons.
- Fabricating benchmark results, citations, or experimental conditions.
- Publishing private or sensitive information without permission.
- Deliberately disruptive spam, trolling, or misrepresentation.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this policy, and may restrict participation when needed to protect the project.

## Reporting

Until the GitHub repository is created, report conduct issues directly to the repository owner through the channel you already use for this workspace. After publication, replace this section with the canonical contact path for the repository.
